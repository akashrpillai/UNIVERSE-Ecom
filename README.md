# UNIVERSE-Ecommerce

Hi! My name is **Akash R Pillai** and this is my Ecommerce Project.
 **UNIVERSE-Ecom** is a modern Ecommerce web application for buying 
 and selling goods online. 

#  Install Dependencies

**For Backend** - `npm i`

**For Frontend** - `cd frontend` ` npm i`
